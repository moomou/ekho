What is Ekho?

Ekho allows anyone to record mouse and keyboard actions and replay them by speaking.

This can be very helpful when working with complex web apps when hands are tied to mouse and keyboard.

FAQ



  • Q. How do I try it out on another page?

    A. In theory, one should be able to try out Ekho on any web page by copying the bonus section snippet from above. There are several things that could go wrong, however. Many times Ekho will load but the UI will not show up properly because of DOM placement on the page. This requires manually figuring out the page DOM and find a suitable container to inject Ekho UI. Finally, any site with content-security-policy set will block Ekho from working properly.

  • Q. Are all interactions supported?

    A. Currently, only left mouse click and keyboard event are supported. Link navigation is not supported.
